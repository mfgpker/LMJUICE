Fixing the ValidateInput function you need to complete the code seen in the challenge. As the description says we need to fix XSS attacks, and hence we need to sanitize our input. If you are experienced in JavaScript you can try to make your own function, or you can simply apply our function named "sanitization"!

Hints:

  • What we call 'input' in our function is an example of an attempted XSS attack which tries to trigger an alert with the text "XSS hallo" on our site.
  • A function can be called by using it's name and then defining what you want to give it.
